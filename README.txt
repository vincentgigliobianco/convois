Instructions pour ouvrir et exécuter le notebook "test_suneris_vgigliobianco.ipynb":

1. Se déplacer dans le dossier "test_suneris_vgigliobianco/data" et dézipper tous les fichiers df1.zip à df6.zip
2. Se déplacer dans le dossier "test_suneris_vgigliobianco"
3. Créer l'environnement Conda "test_suneris_vgigliobianco" avec la commande:
conda env create -f env_test_suneris_vgigliobianco.yml

conda install ipykernel
ipython kernel install --user --name=suneris

Si besoin activer l'environnement avec la commande: conda activate test_suneris_vgigliobianco



4. Faire la commande: jupyter lab ou Jupyter notebook
5. Ouvrir le notebook "test_suneris_vgigliobianco.ipynb"
